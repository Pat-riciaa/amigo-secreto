# Amigo Secreto 🎁

Um simples aplicativo web para sortear amigos secretos!

## 📌 Como Usar
1. Adicione os nomes na lista.
2. Clique em **"Sortear Amigo Secreto"**.
3. O sorteio será feito automaticamente, sem repetir nomes.

## 🛠️ Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
